Github eBook Editor Click to

This is an EPUB3 editor written using just Javascript that stores changes in a github repository using Github's API (gh3).

You can download an EPUB3 of the book by clicking the "Zip" button on the github page for the book.

You can:

  • edit a book by editing the Table of Contents (the entry with a icon) or by clicking an individual HTML page to edit
  • change which book you are editing by clicking the Settings button on the top-right of the editor.
  • fork a book by Signing in and clicking Fork this Book!
  • To see the commits click the link on top of the editor page

Fork the code on github!
